Welcome!

You have found the REST API for the WildFire Exemplar. For more information, see the README on the GitHub page.

Monitoring & Executing

  • You can monitor all current constants and values by
    GET
    -ing
    /monitor
    .
  • You can get the current adaptation option values
    GET
    -ing
    /adaptation_options
    .
  • You can monitor all values by
    PUT
    -ing
    /execute
    .

Schemas

  • You can see monitor schema by
    GET
    -ing
    /monitor_schema
    .
  • You can see adaptation options schema by
    GET
    -ing
    /adaptation_options_schema
    .
  • You can see execute schema by
    GET
    -ing
    /execute_schema
    .
